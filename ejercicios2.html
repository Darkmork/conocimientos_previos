Consideremos tres conjuntos de letras que llamamos conjuntos I, II y III.
I = {a, m, r}
II = {b, d, i, l,u}
III = {c, e, n, t}
¿Cuántos modos hay para elegir una letra de los conjuntos I, II y III?.

Nótese que estos tres conjuntos son disjuntos o mutuamente excluyentes, es decir no hay elementos en común entre ellos.

Para elegir una letra del conjunto I hay 3 maneras, para elegir una letra del conjunto II hay 5 maneras y para elegir una letra del conjunto III hay 4 maneras. Luego para elegir una letra de entre los tres conjuntos hay 12 maneras.
